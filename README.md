# Cultivar

Realistic crop cultivation plugin for Minecraft 1.20.1.

Grow Cannabis, Tobacco, Tea, and Mushrooms with unique growth mechanics, strain genetics, post-harvest processing, and a reputation system.

## Build

```bash
mvn clean package
```

JAR output: `target/cultivar-1.0.0-SNAPSHOT.jar`

## Setup

1. Place JAR in server `plugins` folder
2. Restart server
3. Configure `plugins/Cultivar/config.yml`
4. Grant `cultivar.use` permission to players
